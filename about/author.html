Desktop customizing

I’ve been watching the customizing scene for many years now, going back to the days when Windows 98 was as good as it got. I used to use Litestep and several other programs to do some rather serious customization work, but nowadays I stick to customizing the taskbar with TClock2 and changing the shell icons using IconPhile. You can take a look at my current desktop (or the low-quality version) to see the results.

I have a collection of many different wallpapers that I rotate through. I can’t recall where I found most of them, but I have an old list of sources that you can look through. Probably the best place for wallpapers nowadays is Customize.org. I especially like the work of hermik (who is on Customize.org).

A few programs that I like:

  • Programmer’s Notepad is a very nice, open-source text editor
  • Miranda is a somewhat quirky but very lightweight multi-network instant messenging client
  • PuTTY is an excellent SSH client with a lot of nice features
  • FileZilla is an FTP client that also supports Secure FTP and file transfer over SSH
  • Winamp is still the best music player ever, but you should get version 2.91
  • And of course I use Firefox and Thunderbird
