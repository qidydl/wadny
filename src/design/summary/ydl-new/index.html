September 16, 2000 - 10:43pm

I remembered that I needed to make an update on the site, and since I had some stuff to post anyways, I figured I'll try to remember what I deleted. Anyways, both blink and idtscript have been updated. Blink has a small bugfix that prevents people from putting commands in quit messages and having them executed when you do a seen. IDTScript has a lot more, though, including MP3 updates for mIRC 5.8, new theming aliases, help fixes, and a new theme that's far better than the old default. If you use either of those, I highly recommend you get the new versions, and if you've never tried IDTScript or gave up on it, you should try it now.

The other thing besides script updates concerns blink. As you may have noticed if you run a copy of blink, mIRC does not handle certain things well, if it manages to handle them at all (the EAL starts giving stupid "line too long" errors far too quickly - mIRC has serious limitations). A while ago, I tried to write an irc client/port of blink in C++, but it didn't work too well. Recently, we got a linux computer and I started screwing around with perl. Someone else on Inside3D has a perl bot, and eventually I wound up working on a perl port of blink. Being the incredibly creative (hah) person that I am, it's named "plink". Right now it has a few basic features and some stuff that you can only really do with perl on linux, but I'll eventually get just about everything in there and (hopefully) release it. If you want to see it or whatever, come to irc.inside3d.net #gloom (I'm usually in there as QID|YDL, if not, look for me in #coders).
